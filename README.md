The provided .icmt (transportable database) package includes five separate SQL scripts:
+ NODE
+ CONDUIT
+ SUBCATCHMENT
+ LAND USE
+ RUNOFF SURFACE

These scripts are specifically designed for ICM flag setup in DMT projects, and can be directly applied to projects under Al Ain City Municipality and Abu Dhabi City Municipality.
The core source code structure remains consistent; however, users working on different projects must update the flag values and configurations to match their specific project requirements.
